# gaia-graph-analysis
# Nearest-neighbor graph analysis of Gaia stellar data

This project explores graph-based analysis of stellar position data using a subset of Gaia DR3.

# Objective

Construct nearest-neighbor graphs and analyze structural properties such as: 
- connectivity
- degree distribution
- clustering behavior

# Dataset
Gaia DR3 (subset)

# Tools
-R
-igraph
-FNN / RANN
-ggplot2

# Project Structure
-'data/' -> dataset
-'scripts/' -> R scripts
-'figures/' -> plots
-'notes/' -> drafts and writing
